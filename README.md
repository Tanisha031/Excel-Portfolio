# 📊 Excel Data Analysis Practice Portfolio

## 📌 Repository Description

This repository contains a structured collection of Excel practice datasets, exercises, notes, and project files focused on building strong skills in data analysis using Microsoft Excel. It covers beginner to advanced concepts including data cleaning, formulas, functions, dashboards, pivot tables, charts, and real-world business datasets.

The goal of this portfolio is to demonstrate hands-on experience in Excel for data analysis, reporting, and dashboard creation using industry-relevant datasets from healthcare, sales, finance, operations, and customer analytics domains.

---

## 📁 Folder-wise Overview

---
## 📂 1. Practice Datasets

This folder contains real-world and simulated datasets used for Excel practice and analysis. These datasets are designed to improve data handling, cleaning, and analytical skills.

**Key datasets include:**

- Healthcare (Patients, EHR, Insurance, Patient Monitoring, Churn Data)  
- Sales & Business (Supermarket Sales, Orders, Sales Products, Pizza Sales)  
- IT & Operations (Call Center, Inventory, Employee Management)  
- Media & Services (Netflix, Zomato, Travel Data)  
- Engineering & Equipment (Biomedical Equipment Inventory, Manufacturing Data)  

👉 Purpose: Build analytical thinking using real business scenarios.

---
## 📂 2. Practice Files (My_Work)

This folder includes guided Excel exercises and worksheets focused on foundational to advanced Excel concepts.

**Topics covered:**

- Formulas and Functions (SUMIF, COUNTIF, AVERAGEIF, IF, XLOOKUP)  
- Data Cleaning and Sorting  
- Filtering and Subtotals  
- Pivot Tables and Slicers  
- Charts and Visualization  
- Data Validation and Named Ranges  
- Flash Fill and Shortcuts  

👉 Purpose: Strengthen Excel fundamentals and analytical techniques.
